# relocalization_toolbox

**A 2-D global relocalization framework for LiDAR-only mobile robots**  
Developed on **Ubuntu 20.04** with **ROS Noetic**.

## Overview

A 2-D global relocalization framework that enables LiDAR-only mobile robots to perform initial reliable global pose estimation and recovery without additional sensors. The use of "2D Pose Estimate" in RViz is no longer required.
